## About `_`

The underscore (`_`) at the beginning of the filename (e.g., `_read-zsh-cd-source-code.md`) is used to indicate that the file is not intended to be published by pipeline.


You can set `_`, for example:
- You do not want to publish it yet
- It is WIP (Work In Progress)
