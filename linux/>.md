## Prerequisites

Here is the prerequisite setup for the following examples:

```sh
test_name=redirection_test
tmp_dir=$(date +%y%m%d_%H%M%S_$test_name)
mkdir -p ~/test_dive/$tmp_dir
cd ~/test_dive/$tmp_dir

echo "Target Acquired" > valid_file.txt

cat valid_file.txt
```


## 1> | >: Redirects only stdout


```sh
grep -r '^The' > /etc/ 1> errors.txt
```

## 2>: Redirects only stderr

> [!NOTE]
> Note that the directory `/dev/null` is used to discard unwanted output

```sh
ls -l valid_file.txt ghost_file > output.log
```


## >>: Appends output to a file

```sh
echo "new line" >> file.txt
```

## <: Redirects input from a file

```sh
sort < file.txt
```

## <<: here document

```sh
bc << EOF
scale=2
10/3
EOF
# 3.33
```

## <<<: Here string

```sh
grep 'hello' <<< "hello world"
# hello world
```

```sh
bc <<< "scale=2; 10/3"
# 3.33
```
